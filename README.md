# Project EAS Pemrograman Web

A web-based document and invoice management system developed using Laravel.

## Features

- Authentication (Login & Register)
- Profile Management
- Berita Acara Management
- Invoice Management
- Role-based Access Control
- Payment Status Monitoring

## Technologies

- Laravel 10
- PHP
- MySQL
- Tailwind CSS
- Laravel Breeze

## Installation

```bash
git clone <repository-url>
cd project-eas-pemweb

composer install
npm install

copy .env.example .env

php artisan key:generate
php artisan migrate
php artisan serve
```
